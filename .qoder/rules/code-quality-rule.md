## Code Qualität Rule

- Eine Funktion = Eine Aufgabe
- Maximale Funktionslänge: 14 Zeilen (HTML ausgenommen). Wichtig!
- Max 400 LOCs pro Datei
- 1-2 Leerzeilen Abstand zwischen Funktionen
- JSDoc Dokumentation für alle Funktionen
